List Error Orangepi

1. Could not chdir to home directory /home/insun: No such file or directory
Pesan itu artinya folder home kamu tidak ditemukan di server OrangePi. Ini terjadi karena user insun terdaftar di sistem tapi folder /home/insun nya belum ada atau terhapus.

Cara Mengatasinya
Langkah 1 — Cek Apakah Folder Ada
ls /home
Lihat apakah folder insun ada atau tidak.

Langkah 2 — Buat Folder Home
Jika foldernya tidak ada, buat manual:
sudo mkdir /home/insun

Langkah 3 — Set Kepemilikan Folder
sudo chown insun:insun /home/insun

Langkah 4 — Set Permission Folder
sudo chmod 755 /home/insun

Langkah 5 — Buat Ulang Folder Home Otomatis
Cara lebih lengkap, biarkan sistem yang buat otomatis:
sudo mkhomedir_helper insun

Setelah Itu Logout & Login Ulang
exit

Lalu login kembali, pesan error tersebut seharusnya sudah hilang.

Kenapa Ini Bisa Terjadi?
- Folder terhapusFolder /home/insun tidak sengaja terhapus
- User dibuat manualUser dibuat tanpa otomatis membuat folder home
- Reinstall OSOS diinstall ulang tapi user lama masih terdaftar

2. ntfs-3g-mount: bad mount point home/insun/media/insundrive: No such file or directory
Error ini artinya folder tujuan mount harddisk belum ada. Folder /home/insun/media/insundrive harus dibuat dulu sebelum bisa di-mount.

Langkah 1 — Buat Folder Mount Point
sudo mkdir -p /home/insun/media/insundrive
-p artinya buat semua folder sekaligus meskipun folder parent belum ada

Langkah 2 — Set Kepemilikan Folder
sudo chown insun:insun /home/insun/media/insundrive

Langkah 3 — Coba Mount Ulang
sudo mount /dev/sdX /home/insun/media/insundrive

Ganti /dev/sdX dengan nama harddisk kamu, cek dengan perintah:
lsblk

Langkah 4 — Cek Apakah Berhasil
ls /home/insun/media/insundrive
Jika isi harddisk muncul berarti berhasil ✅
