Kumpulan Insight

1. Auto mount hdd
Menggunakan /etc/fstab (Direkomendasikan)

Langkah 1 — Cari UUID Harddisk
bashsudo blkid
Output contoh:
/dev/sda1: UUID="1234-5678" TYPE="ntfs"
Catat UUID harddiskmu.

Langkah 2 — Buka File fstab
sudo nano /etc/fstab

Langkah 3 — Tambahkan Baris Ini
UUID=1234-5678  /home/insun/media/insundrive  ntfs-3g  defaults,auto,users,rw,nofail  0  0
Penjelasan:
BagianArtiUUID=1234-5678Identitas harddisk kamu/home/insun/media/insundriveFolder mount pointntfs-3gFormat harddisk (ntfs)defaults,autoMount otomatis saat bootnofailServer tetap jalan meski harddisk tidak terbaca

Langkah 4 — Test Tanpa Reboot
sudo mount -a

Langkah 5 — Cek Hasilnya
lsblk

2. Install PhpMyadmin
Baik, karena pakai Nginx prosesnya sedikit berbeda! Berikut langkah-langkahnya:

Langkah 1 — Update Sistem
bashsudo apt update && sudo apt upgrade -y

Langkah 2 — Install PHP & PHP-FPM (Jika Belum Ada)
Nginx menggunakan PHP-FPM bukan mod_php seperti Apache:
bashsudo apt install php php-fpm php-mysql php-mbstring php-zip php-gd php-json php-curl -y

Langkah 3 — Install phpMyAdmin
bashsudo apt install phpmyadmin -y
Saat instalasi muncul pertanyaan:
1. Pilih web server → JANGAN pilih apapun (kosongkan, langsung Enter)
   karena kita pakai Nginx
2. Configure database with dbconfig-common? → pilih "Yes"
3. Masukkan password MySQL kamu

Langkah 4 — Buat Symbolic Link ke Nginx
bashsudo ln -s /usr/share/phpmyadmin /var/www/html/phpmyadmin

Langkah 5 — Konfigurasi Nginx
bashsudo nano /etc/nginx/sites-available/phpmyadmin
Isi dengan konfigurasi berikut:
nginxserver {
    listen 80;
    server_name 192.168.18.24;  # ganti dengan IP OrangePi kamu

    root /var/www/html;
    index index.php index.html;

    location /phpmyadmin {
        root /var/www/html;
        index index.php;

        location ~ \.php$ {
            include snippets/fastcgi-php.conf;
            fastcgi_pass unix:/var/run/php/php-fpm.sock;
        }
    }
}

Langkah 6 — Aktifkan Konfigurasi Nginx
bash# Aktifkan site
sudo ln -s /etc/nginx/sites-available/phpmyadmin /etc/nginx/sites-enabled/

# Test konfigurasi nginx
sudo nginx -t

# Restart Nginx
sudo systemctl restart nginx

Langkah 7 — Cek Versi PHP-FPM
Pastikan versi PHP-FPM sesuai:
bash# Cek versi PHP
php -v

# Sesuaikan path fpm di konfigurasi nginx
# Contoh jika PHP 8.1
fastcgi_pass unix:/var/run/php/php8.1-fpm.sock;

Langkah 8 — Buat User MySQL
bash# Masuk MySQL
sudo mysql

# Buat user
CREATE USER 'insun'@'localhost' IDENTIFIED BY 'password_kamu';

# Beri akses
GRANT ALL PRIVILEGES ON *.* TO 'insun'@'localhost';

# Simpan
FLUSH PRIVILEGES;

# Keluar
EXIT;

Langkah 9 — Akses phpMyAdmin
Buka browser:
http://192.168.18.24/phpmyadmin
